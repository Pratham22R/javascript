Array

It is a type of object used to store multiple values in a single variable.

It is a non-primitive type.

It is used to store values of same datatype.

Unlike object literal it does not have key value pairs in it.

How to define an array

const array_name = [] (empty array)

var array_name = [value1,value2,value3] (array with values)
