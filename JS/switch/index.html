Switch statement

It is a conditional statement that is used to execute a block of code based on the value of a variable

It is an alternative to if-else.

In this if one of the condition gets satisfied then rest all the below code gets executed.

In order to to just execute the statisfied case only which matches first we use break keyword.

Default keyword is used when we want js to read a particular command if none of the case is acccepted by switch.

Switch is type sensitive it does not convert any data type accoring to the requirements and matches only those cases that has strictly equal value and datatype.

Syntax

switch(expression) {

case x:

break;

case y:

break;

default:

}
