Conditional statement

  1. If
  2. Else if
  3. Else

If

It is a Conditional statement that is used to execute a block of code if a condition is true

Else If

it is a conditional statement that is used to execute when the if condition is false

Else

It is a Conditional statement that is used to execute a block of code when all the conditions are false
